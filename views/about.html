Lazy Game v1.0 User Guide

Lazy Game é um jogo simples onde eu, Rafael Luiz Rehn, aproveito a ferramentas Angular, JavaScript e PHP para aplicar algumas funcionalidades teste para aplicações web



Como jogar?

É muito Facil! Basta clicar no botão Start.


E depois?

Bem, depois um contador de tempo é ativado enquanto o botão é mantido pressionado.


Conta pontos?

Não conta pontos, mas o seu maior tempo é armazenado e você pode registrar seu resultado no ranking.


Qual ranking?

Pode-se conferir o rankinkg no menu superior Ranking.
